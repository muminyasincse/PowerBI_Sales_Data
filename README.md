# Mobile Sales Dashboard | Power BI

## Overview
This project analyzes mobile phone sales data using Microsoft Power BI.
The dashboard converts raw sales data into clear, actionable insights across
products, time, customer segments, payment methods, and cities.

## Key KPIs
- Total Sales Revenue
- Total Quantity Sold
- Number of Transactions
- Average Sales Value

## Dashboard Analysis
- Sales performance by mobile brand and model
- Monthly quantity trends
- Sales distribution by day of week
- Payment method contribution
- Customer rating breakdown
- City-wise sales performance across India

## Tools & Technologies
- Microsoft Power BI
- DAX Measures
- Data Modeling
- Excel (Data Source)

## Files Included
- `Mobile_Sales_Dashboard_PowerBI.pbix` – Power BI dashboard file
- `Mobile_Sales_Data.xlsx` – Raw sales dataset
- `dashboard_Sales_Data.png` – Dashboard screenshot
